Module 1: Introduction to the topic, the course, and NLP

Module 1 will first introduce the main topic of the course, the media coverage of natural and environmental disasters - including possible research topics - in German newspapers and give an overview about the course requirements.

Natural Language Processing (NLP), which has become increasingly crucial in our data-driven world, will be introduced. This includes core linguistic concepts relevant to NLP, basic NLP tasks and their applications as well as current challenges and future directions in the field.

As programming language we will use Python. Module 1 includes a crash course in Python, covering basic syntax and data types, Control structures (if statements, loops) and data structures (lists, dictionaries).

  • Open Google Colab Documentation (also under "Important Links" on the main course page) and complete steps 1 to 4.
